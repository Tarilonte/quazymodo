modalComponent

Use quando a pessoa precisa tomar uma decisao clara ou confirmar uma acao importante sem distracoes.

Instalacao

Inclua o plugin no blueprint da pagina: 

ComponentFactory::Plugin(componentName: '/plugins/modalComponent/')

Como usar

  • Renderize o plugin uma unica vez na pagina.
  • Abra o modal com titulo e mensagem de contexto.
  • Use o modo travado em fluxos criticos temporarios.

Metodos 

modal_show(header, message, addClass, locked)
modal_close()
modal_lock()
modal_unlock()

Exemplo

Codigo usado neste card: 

<button id="btn-exemplo-modal" class="btn btn-primary">Abrir modal</button>
$('#btn-exemplo-modal').on('click', function () {
  modal_show('Confirmacao orientada', 'Use modal para acao focada.', false, false);
});
