toastComponent

Ideal para respostas rapidas: sucesso, aviso ou erro sem bloquear o fluxo principal da pagina.

Instalacao

Inclua o plugin no blueprint da pagina: 

ComponentFactory::Plugin(componentName: '/plugins/toastComponent/')

Como usar

  • Renderize o plugin uma unica vez no layout.
  • Defina uma mensagem curta e tipo semantico.
  • Ajuste duracao conforme urgencia do feedback.

Metodos 

ToastComponent.newToast(message, duration, toastType)

Exemplo

Codigo usado neste card: 

<button data-toast-type="success" class="btn btn-sm btn-success">Success</button>
$('[data-toast-type]').on('click', function () {
  const toastType = $(this).data('toast-type');
  ToastComponent.newToast('Feedback rapido e nao bloqueante.', 4200, toastType);
});
