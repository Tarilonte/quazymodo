verticalTable

Organiza pares chave-valor com leitura rapida. Bom para config, estado e inspecao de payloads.

Instalacao

Inclua o shortcut no blueprint: 

use App\Components\ComponentShortcuts as UI;
UI::verticalTable(rows: $dados, options: ['th-class' => 'text-primary'])

Como usar

  • Passe os dados como array associativo em rows.
  • Ajuste th-class e td-class para estilo.
  • Prefira para metadados e payloads de depuracao.

Metodos 

UI::verticalTable(rows: array, options: array = [])

Exemplo

Codigo usado neste card: 

'vertical-table-demo' => UI::verticalTable(
  rows: ['status' => 'ativo', 'dependencias' => 'Tailwind + daisyUI'],
  options: ['table-class' => 'table-zebra bg-base-100 rounded-box']
)
